# imageTCGAWorkflow: Workflow for Histopathology Image Analysis with TCGA Data

A Bioconductor workflow demonstrating end-to-end histopathology image
analysis using TCGA diagnostic whole-slide images. See the package
vignette for the complete workflow.

## See also
